- Annotator
  查看 git 提交记录
- Code Outline
  展示代码结构树
- Git Blame
  在状态栏显示当前行的 Git 信息
- GitLens
  显示文件最近的 commit 和作者，显示当前行 commit 信息

- Git History(git log)
  查看 git log
- HTML CSS Support
  css 提示（支持 vue）
- Guides
  高亮缩进基准线
- Prettify JSON
  格式化 JSON

- Prettier - Code formatter
  代码格式化
- SCSS IntelliSense
  scss 智能提示

- TSLint
  TSLint
- TypeScript Importer
  TypeScript Importer
- vscode-icons
  图标
